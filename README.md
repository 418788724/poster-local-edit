# EPOSTER 离线海报制作工具

完全本地的海报制作工具，基于[fabic.js](https://github.com/fabricjs/fabric.js)和[umi](https://github.com/umijs/umi)无需依赖任何服务端，所有数据都存储在本地。

## 功能

-   [x] 海报设计
-   [x] 海报导出
-   [ ] 海报分享
-   [ ] 海报模板
-   [ ] more...

## 安装

```
    pnpm install
```

本地启动

```
    pnpm run start:dev
```

部署

```
    pnpm run build
```
